# Python To Do Application 

This project is a command line To-Do List application built using Python. Allows users to add, view, and delete tasks through a simple menu. 
The program runs in a loop until the user decides to quit. Includes exception handling to ensure the program doesn't break due to invalid input.

## How To Run 

Clone the repository and navigate to the folder to run index.py. To use the application enter the corresponding number according to the menu.
